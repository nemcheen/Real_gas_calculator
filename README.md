Real Gas Calculator (Python)
Небольшой консольный проект на Python для выбора газа из списка и получения его параметров из словаря gases.
Отрисовки графика коэффициента сжимаемости. 
Расчет всех парамтеров уравнения газа для текущей Т и P

Структура проекта
main.py — точка входа
gas_params.py — словарь gases с параметрами газов
gas_io.py — функции ввода/проверок
plot.py - отрисовка 3D графика Z(P, T)
calc.py - функции вычисления всех парамтеров в том числе Z

Пример словаря:
gases = {
    'metan': (190.6, 4.6e6, 0.011),      # метан
    'azot': (126.2, 3.39e6, 0.037),      # азот N₂
    'kislorod': (154.6, 5.04e6, 0.022),  # кислород O₂  
    'uglerod_oxid': (304.2, 7.39e6, 0.042),  # углекислый газ CO₂
    'etanol': (240.8, 6.14e6, 0.152),    # этан C₂H₆ 
}

Копировать код

git clone https://github.com/<your-username>/<your-repo>.git
cd <your-repo>

Запуск (предварительно установив зависимости из requiremetns.txt)
python main.py

Использование
После запуска программа покажет консольное меню
Расчитать параметры реального газа: [C]
Вывести 3D график Z(P, T) [Z]
Выход [E]

Вводите требуемые значения и получайте результат расчетов.
